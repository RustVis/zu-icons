## Usage

Add a dependency to your `Cargo.toml`:

```toml
[dependencies]
dioxus-icons-lucide = "0.2"
```

Then use the icons in your Dioxus components:

```rust
use dioxus::prelude::*;
use dioxus_icons_lucide::{ZoomOut, Icon};

fn App() -> Element {
    rsx! {
        Icon {
            icon: ZoomOut,
            size: "24px",
            stroke_width: 1.5,
        }
    }
}
```

All icon packs use the same `Icon` component from `dioxus-icon-component`, ensuring a consistent API across every set.
